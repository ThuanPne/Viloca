# Travel Planning Mobile App - Feature Specification

## 1. Core Features

### 1.1 Trip Planner
Cho phép người dùng tạo và quản lý các chuyến đi.

**Chức năng:**
- Tạo chuyến đi mới.
- Thiết lập điểm đến.
- Chọn ngày bắt đầu và kết thúc.
- Quản lý lịch trình theo từng ngày.
- Sắp xếp hoạt động bằng kéo thả.

---

### 1.2 Budget Management
Theo dõi ngân sách và chi tiêu của chuyến đi.

**Chức năng:**
- Nhập ngân sách dự kiến.
- Ghi nhận chi phí thực tế.
- Phân loại chi phí:
  - Di chuyển
  - Khách sạn
  - Ăn uống
  - Vé tham quan
  - Mua sắm
- Thống kê tổng chi phí.
- Hiển thị ngân sách còn lại.

---

### 1.3 Packing List
Quản lý danh sách vật dụng cần mang theo.

**Chức năng:**
- Tạo checklist cá nhân.
- Đánh dấu hoàn thành.
- Gợi ý danh sách theo loại chuyến đi:
  - Du lịch biển
  - Du lịch núi
  - Công tác
  - Du lịch quốc tế

---

### 1.4 Route & Map Management
Quản lý hành trình trên bản đồ.

**Chức năng:**
- Hiển thị các điểm đến trên bản đồ.
- Xem khoảng cách giữa các địa điểm.
- Ước tính thời gian di chuyển.
- Tối ưu tuyến đường.

---

### 1.5 Booking Management
Lưu trữ và quản lý thông tin đặt chỗ.

**Chức năng:**
- Lưu vé máy bay.
- Lưu vé tàu xe.
- Lưu thông tin khách sạn.
- Nhắc nhở giờ khởi hành.
- Nhắc nhở check-in/check-out.

---

### 1.6 Smart Destination Recommendation
Gợi ý địa điểm phù hợp với người dùng.

**Chức năng:**
- Gợi ý điểm tham quan.
- Gợi ý nhà hàng.
- Gợi ý quán cà phê.
- Gợi ý sự kiện địa phương.
- Lọc theo sở thích:
  - Văn hóa
  - Thiên nhiên
  - Ẩm thực
  - Check-in

---

### 1.7 Group Trip Planning
Hỗ trợ lập kế hoạch du lịch nhóm.

**Chức năng:**
- Mời thành viên tham gia.
- Chia sẻ lịch trình.
- Đồng bộ dữ liệu theo thời gian thực.
- Bình chọn địa điểm.
- Chia sẻ chi phí.

---

### 1.8 Travel Journal
Lưu giữ kỷ niệm chuyến đi.

**Chức năng:**
- Viết nhật ký.
- Đính kèm hình ảnh.
- Đính kèm video.
- Ghi chú cảm xúc.
- Xem lại theo dòng thời gian.

---

### 1.9 Weather & Travel Alerts
Cập nhật thông tin thời tiết và cảnh báo.

**Chức năng:**
- Dự báo thời tiết.
- Cảnh báo mưa.
- Cảnh báo bão.
- Thông báo các sự kiện ảnh hưởng đến chuyến đi.

---

### 1.10 AI Travel Assistant
Trợ lý AI hỗ trợ lập kế hoạch du lịch.

**Chức năng:**
- Tạo lịch trình tự động.
- Tối ưu chi phí.
- Đề xuất địa điểm.
- Đề xuất phương tiện di chuyển.
- Trả lời các câu hỏi liên quan đến chuyến đi.

## Feature 3 — Wishlist / Bookmark

### Database migration

**`supabase/migrations/20260620000000_add_bookmarks.sql`**

```sql
create table public.bookmarks (
  id            uuid primary key default gen_random_uuid(),
  user_id       uuid references public.profiles(id) on delete cascade not null,
  experience_id text not null,
  status        text check (status in ('want', 'planned', 'done')) default 'want',
  created_at    timestamptz default now(),
  unique(user_id, experience_id)
);

alter table public.bookmarks enable row level security;

create policy "Users manage own bookmarks"
  on public.bookmarks for all
  using  (auth.uid() = user_id)
  with check (auth.uid() = user_id);
```

## Feature 4 — Packing List

### Database migration

**`supabase/migrations/20260620000001_add_packing_items.sql`**

```sql
create table public.packing_items (
  id         uuid primary key default gen_random_uuid(),
  trip_id    uuid references public.trips(id) on delete cascade not null,
  name       text not null,
  is_packed  boolean default false,
  category   text default 'other',
  sort_order int default 0,
  created_at timestamptz default now()
);

alter table public.packing_items enable row level security;

create policy "Users manage packing via trip ownership"
  on public.packing_items for all
  using (
    exists (select 1 from public.trips t where t.id = trip_id and t.user_id = auth.uid())
  );
```
